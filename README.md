# Проект: Library System (Библиотечна система)

## Описание

Този проект представлява малка C++ библиотечна система, изградена от няколко взаимосвързани класа — **Author, Book, Member, Loan и Library**.

Всички класове са реализирани **изцяло в .h файлове** (header-only архитектура).

Проектът демонстрира ключови концепции от ООП:

- конструктори и списъци за инициализация  
- Rule of 5 / Rule of 3 (= default, когато е подходящо)  
- капсулация и валидация на данни  
- статични членове  
- const-коректност  
- работа с `std::vector` и обекти  
- разделение на модулите чрез header-only дизайн  

---

```
library/

├── Loan.h

```

---

## Компилация и изпълнение

Влезте в директорията на проекта и изпълнете:

```
g++ -std=c++17 -Wall -Wextra -O2 main.cpp -o library
./library
```

Тъй като проектът е **header-only**, няма нужда от компилиране на отделни .cpp файлове.

## Класове

### **Клас Author**

Представя автор на книга.

**Членове:**
- `string name`
- `int birthYear`

**Характеристики:**
- конструктор по подразбиране  
- параметризиран конструктор с валидация  
- сетъри с проверки  
- `to_string()` метод  

---

### **Клас Book**

Представя книга.

**Членове:**
- `string title`
- `Author author`
- `int year`
- `double price`
- `string isbn`
- `static int totalBooks`

**Характеристики:**
- Rule of 5 (копиране/преместване = default)  
- валидации за година и цена  
- `to_string()`  
- статичен брояч на книгите  

---

### **Клас Member**

Представя читател.

**Членове:**
- `string name`
- `string memberId`
- `int yearJoined`

**Функции:**
- параметризиран конструктор с валидация  
- `to_string()`  

---

### **Клас Loan**

Представя заем на книга.

**Членове:**
- `string isbn`
- `string memberId`
- `string startDate`
- `string dueDate`
- `bool returned`

**Функции:**
- валидация на датите  
- `markReturned()`  
- `isOverdue()`  
- `to_string()`  

---

### **Клас Library**

Управлява цялата библиотечна система.

**Членове:**
- `vector<Book> books`
- `vector<Member> members`
- `vector<Loan> loans`

**Методи:**
- добавяне на книги  
- добавяне на читатели  
- заемане и връщане на книги  
- проверка за наличност  
- търсене по автор  
- системна информация (`to_string()`)  

---

## main.cpp

Демонстрира използването на системата:

- създаване на обекти  
- добавяне в библиотеката  
- заемане и връщане на книги  
- търсене по автор  
- статистика
